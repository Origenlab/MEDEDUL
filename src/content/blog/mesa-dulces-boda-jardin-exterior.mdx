## 1. Planeacion exterior: la base de todo

### El error que mas se repite

Montar como si fuera salon cerrado. Esto provoca problemas de conservacion, flujo de servicio y estabilidad visual a mitad del evento.

### Enfoque profesional en exterior

Antes de definir surtido, evaluamos orientacion solar, sombras, viento, humedad, acceso de montaje y tiempos de mayor consumo. Esa lectura tecnica determina diseno y operacion.

### Variables que debes considerar

- Horario real de apertura de mesa.
- Distancia entre cocina, barra y zona de invitados.
- Superficie del terreno y nivelacion de mobiliario.
- Posibles cambios de clima en la franja del evento.

## 2. Costos orientativos para boda en jardin

Para mejorar decision comercial:

- Desde $4,500 MXN: formato base exterior.
- Desde $7,900 MXN: opcion intermedia con mejor cobertura.
- Desde $12,500 MXN: premium exterior con operacion robusta.

Que mueve mas el precio:

- numero de invitados,
- nivel de personalizacion,
- complejidad de traslado y montaje,
- necesidad de plan B,
- tiempo de permanencia operativa.

## 3. Surtido recomendado para mayor estabilidad

Para exterior suelen funcionar mejor:

- mini postres de estructura firme,
- chocolates con manejo termico adecuado,
- formatos de porcion individual,
- mix premium con reposicion programada.

Evita surtidos muy sensibles al calor si no hay control de sombra o ventilacion adecuada.

## 4. Errores comunes en mesas de boda exterior

### Error 1: ubicar mesa en sol directo

Afecta textura, sabor y presentacion.

### Error 2: no reforzar estructura

Viento y terreno irregular comprometen seguridad.

### Error 3: abrir mesa sin tiempos claros

Se concentra consumo y baja la experiencia.

### Error 4: no tener plan de contingencia

Un cambio de clima puede detener todo si no hay alternativa.
